# Number Statistics Calculator 📊

A simple console-based Python application that accepts multiple numbers from the user and calculates useful statistics.

## Features

- Accepts multiple numbers from the user
- Calculates the sum of all numbers
- Finds the average value
- Identifies the largest number
- Identifies the smallest number

## Concepts Used

- Variables
- Input and Output
- For Loops
- If-Else Statements
- Arithmetic Operations

## Technologies

- Python 3

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory

```bash
cd number-statistics-calculator
```

3. Run the program

```bash
python number_statistics.py
```
