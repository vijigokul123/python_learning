# List Operations in Python

## Description
This program performs two operations on a list:
- Reverses the list.
- Finds the largest element without using the max() function.

## Concepts Used
- Lists
- List Slicing
- For Loop
- Conditional Statements

## Sample Output

Original list: [10, 45, 23, 89, 12]
Reversed list: [12, 89, 23, 45, 10]
Largest element: 89
