# 🛒 Smart Shopping Bill Calculator

A Python-based billing application that calculates the total purchase amount, applies membership discounts, and generates a detailed bill summary.

---

## 🚀 Features

* Customer information input
* Product and quantity management
* Automatic bill calculation
* Premium member discount support
* Final amount generation
* Formatted bill summary

---

## 📋 Inputs Required

* Customer Name
* Item Name
* Price per Item
* Quantity
* Membership Type

---

## 🛠️ Technologies Used

* Python 3
* Visual Studio Code
* Git
* GitHub

---

## ▶️ How to Run

```bash
python billcalculator.py
```

---

## 📸 Sample Input

```text
Customer Name: Gokul R
Item Name: Headphones
Price per Item: 1500
Quantity: 2

Membership Options
1. Premium Member
2. Regular Customer

Choose Option: 1
```
