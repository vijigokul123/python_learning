# 🐍 Personal Profile Generator

A beginner-friendly Python project that collects personal information from the user and displays a formatted profile summary.

---

## 🚀 Features

- Accepts user input from the console
- Demonstrates variables and data types
- Performs a simple birth year calculation
- Displays a clean profile summary
- Beginner-friendly Python project

---

## 📋 Information Collected

- Name
- Age
- Gender
- City
- College
- Favorite Programming Language

---

- Python 3
- Visual Studio Code
- Git
- GitHub
