# 🐍 Personal Profile Generator

A simple Python program that collects personal information from the user and displays a formatted profile summary.

## 🚀 Features

* Takes user input from the console
* Stores data using variables
* Demonstrates Python data types and input/output operations
* Calculates the user's birth year from their age
* Displays a clean profile summary

## 📋 Information Collected

* Name
* Age
* Gender
* City
* College
* Favorite Programming Language

## 🛠️ Technologies Used

* Python 3
* Visual Studio Code
* Git
* GitHub

## ▶️ How to Run

```bash
python personalprofile.py
```
